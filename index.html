Data tables used to generate data lists from this page

Rare Natural Elements of Georgia: A CSV file of rare species and natural communities tracked by Georgia DNR, Nongame Conservation Section. This file can be imported into almost any database, spreadsheet or GIS program.

Rare Natural Element Occurrences (EOs) of Georgia: A CSV file of rare species and natural community locations by county, HUC10 watershed and other unit areas . This file can be imported into almost any database, spreadsheet or GIS program.
