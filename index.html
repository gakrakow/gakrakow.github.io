Download CSV data tables used to generate element lists in this data portal

These CSV (comma separated value) files can be imported into virtually all database, spreadsheet or GIS programs created in the last 20 years including Excel, Access, ArcGIS, QGIS and many more.

es_web.csv - Rare Natural Elements of Georgia - Rare species and natural communities tracked by Georgia DNR, Nongame Conservation Section.

eo_web.csv - Locations of Rare Natural Element of Georgia - Rare species and natural community element occurrences (EOs) by county, HUC10 watershed and other unit areas with shapefile download links below.

es_vw.csv - Database query view used to generate element lists - The CartoDB/PostGIS database SQL query view used for generating element lists. This query view joins information from es_web, eo_web and several other tables with information used for generating rare element lists in this data portal.

Download index shapefiles used to do analysis and make maps for this data portal

These ESRI shapefiles are exported from the geospatial data layers used to join and filter rare element data used in this data portal. The coordinate reference system is WGS84 (epsg:4326), also know as World Geodetic System of 1984 or just latitude/longitude.

cnty.shp - Georgia Counties

qq.shp - Quarter quads - One fourth of a USGS 7.5 minute quadrangle map

huc8.shp - USGS Hydrological Unit Code 8 Digit Watersheds

huc10.shp - USGS Hydrological Unit Code 10 Digit Watersheds

eco3.shp - EPA Level III Ecoregions

eco4.shp - EPA Level IV Ecoregions

hex24.shp - 24 km Hexagon Grid
